/kev Usage
  • /kev - Get all Known Exploited Vulnerabilities
  • /kev?search=Microsoft - Search KEV based on description
  • /kev/CVE-ID - Fetch a KEV by CVE-ID
  • /kev/recent?days=7 - Get new vulnerabilities added in the last 7 days (modifiable)
  • /kev?sort=severity&order=desc/asc - Sort all KEV by NVD baseScore (if it has one). You can even add &search= ;)
/vuln Usage
  • /vuln/CVE-2023-1234 - Get all data on a CVE (CISA, MITRE, NVD)
  • /vuln/CVE-2023-1234/nvd - Get all NVD data on a CVE
  • /vuln/CVE-2023-1234/mitre - Get all Mite/Cve.org data on a CVE
Limits
  • KEVin is currently set to allow one (1) request per second. A one (1) hour block is placed on IPs violating this limit.
  • Edge cache is set to 3 hours. Updates may be delayed if CVE data is stuck in cache
  • As of 29 August 2023, search is limited to only KEV data
