About KEVin API

Known Exploited Vulnerabilities in an API (KEVin) is an API that provides vulnerability CVE data from CISA's Known Exploited Vulnerabilities database. KEVin is developed and maintained by Synfinner.

From Synfinner: I created KEVin to help the common security professional/researcher or small business that could not always afford the higher costs of other vulnerability CVE APIs that had some level of threat intelligence association. I was always looking for an API that could be used for us small-time users or companies that just needed the data without all the overhead of other API service providers. Not to mention somewhat loose usage restrictions. I hope that you've found KEVin to be useful in your work.
